Варіант №16

1) Найти наибольшую и наименьшую цифры в записи данного натурального числа.

2) Произведение n первых нечетных чисел равно p. Сколько сомножителей взято? Если введенное число n не является указанным произведением, сообщить об этом.

3) Найти на отрезке [n; m] натуральное число, имеющее наибольшее количество делителей.

4) Задумано некоторое число x (x < 100). Известны числа k, m, n — остатки от деления этого числа на 3, 5, 7. Найти x.
